# 아키텍처 개요

이 문서는 포트폴리오 프론트와 RAG 챗(Cloud Run)을 연결하는 상위 구조를 요약한다.

## 구성
- 프론트: 단일 페이지 포트폴리오(UI)
- 백엔드: RAG 챗 API(Cloud Run)
- 문서 근거: `reference/` 내 Markdown 기반 지식

## 흐름
1) 사용자가 프론트에서 질문 입력  
2) 프론트가 Cloud Run API 호출  
3) 백엔드가 `reference/` 문서를 검색/요약  
4) 답변과 출처를 반환  

## 원칙
- 답변은 `reference/` 근거에 기반
- 근거가 없으면 “수정중/제작중”으로 응답
